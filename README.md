# 🧠 VitaLoop

### AI 기반 비침습 질환 위험 예측 및 행동 유도형 건강관리 플랫폼

---

## 📌 프로젝트 개요

**VitaLoop**는 체지방계(BIA) 및 생활습관 데이터를 기반으로

AI를 활용하여 대사질환 위험도를 예측하고,

사용자의 건강 행동을 지속적으로 유도하는 예방 중심 헬스케어 시스템이다.

기존 건강검진은 병원 방문 기반의 주기적 검사 방식으로

일상적인 건강 변화 반영이 어렵고 조기 예측에 한계가 있다.

VitaLoop는

👉 **AI 질환 예측 + 행동 유도 피드백 시스템**을 결합하여

개인의 일상 속에서 지속 가능한 건강관리를 가능하게 한다.

---

## 🚨 문제 정의

- 대사질환 유병률 증가 (성인 3명 중 1명)
- 병원 중심 검사 → 접근성 낮음
- 검사 주기 길어 데이터 연속성 부족
- 기존 BIA 장비 → 단순 수치 제공

👉 **데이터는 존재하지만 활용되지 못하고 있음**

---

## 🎯 프로젝트 목표

1. AI 기반 질환 위험 예측 모델 구축
2. 의료 데이터 구조화를 위한 데이터베이스 설계
3. 행동 유도형 피드백 시스템 구현 (핵심 기능)

---

↓

[Backend API]

↓

↓

---

## 🧩 주요 기능

### 1. AI 질환 예측

- 모델: XGBoost
- 예측 질환:
    - 고혈압
    - 당뇨
    - 이상지질혈증
    - 대사증후군
    - 비만

---

### 2. 데이터 전처리

- 결측치 처리
- 이상치 제거
- SMOTE (데이터 불균형 해결)
- StandardScaler (정규화)

---

### 3. 행동 유도 피드백 시스템 ⭐ (핵심)

VitaLoop의 핵심 기능으로,

AI 예측 결과를 기반으로 사용자 맞춤형 건강 행동을 유도한다.

### 📌 주요 기능

- 개인화 메시지 생성
- 건강 습관 유지 Streak 시스템
- 위험도 기반 알림 전략

### 💬 예시 메시지

- "오늘 15분만 걸어보는 건 어때요?"
- "최근 활동량이 감소했습니다."
- "현재 당뇨 위험이 높습니다."

## 🔄 데이터 흐름

데이터 수집 → DB 저장 → 전처리 → AI 예측 → 피드백 생성 → 알림 → 사용자 행동 → 데이터 축적 → 모델 개선

👉 **Closed-loop 구조 기반 지속적 개선**
